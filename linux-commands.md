# Developer Commands Cheat Sheet

A simple reference for commonly used Linux commands.

---

## Linux Commands

```bash
# create a new folder
mkdir myFolder

# create a nested folder
mkdir -p myFolder/nestedFolder

# create a new file
touch myFile.txt

# open file in nano editor
nano myFile.txt

# list files in the folder
ls -la

# display the content of the file
cat sample.txt

```
